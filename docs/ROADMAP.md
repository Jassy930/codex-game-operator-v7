# Roadmap

Roadmap is milestone-level only. It is not a task list.

## M0: Harness Ready

- v7.2 docs exist
- goal prompt exists
- governance docs exist
- ops tools exist

## M1: Playable Local MVP

- React + TypeScript + Vite + Vitest
- click resource
- automatic production
- upgrades
- save/load

## M2: Public Preview

- GitHub Pages or Vercel
- release log
- basic feedback entry

## M3: Feedback Processing

- issue ledger
- feedback clusters
- response budget
- no repeated issue replies

## M4: Operated Game

- updates based on clusters
- metrics or manual feedback
- periodic retrospective

## M5: Sustainable Evolution

- richer systems only after core loop proves clear
- harness evolves safely through meta-governance
