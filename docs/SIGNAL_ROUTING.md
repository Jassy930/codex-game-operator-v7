# Signal Routing

GitHub Issues are signals, not tasks.

Default behavior:

```text
Do not implement.
Do not reply.
Do not promise.
First classify.
```

## Classes

### CRITICAL

Immediate action allowed: game does not start, build/deploy broken, save corruption, security/privacy problem.

### ACTIONABLE

Can become a task if it matches current stage and North Star: first upgrade too slow, unclear next goal, too much UI noise, save not working.

### ARCHIVE

Useful later, not now: prestige system, complex skill tree, multiple worlds, advanced automation.

### IGNORE / DECLINE

Does not fit North Star: PvP, multiplayer economy, unrelated platform requests.

## Mainline Protection

No issue can directly modify `ROADMAP.md`.

It must first appear in:

1. `ISSUE_LEDGER.md`
2. `FEEDBACK_CLUSTERS.md`
3. `DECISION.md`
