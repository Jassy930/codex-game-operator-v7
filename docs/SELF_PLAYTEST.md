## 最新结果

2026-05-08 v0.7 `归航航线` 第一段已可玩。这不是真实玩家反馈；触发信号来自用户要求积极扩展玩法内容到 20 小时，以及 v0.6 余辉只能解释新一轮起步价值、还缺少重复归航后的长期方向。

- 已完成：达到 25 台自动采集器和 15 次调校后，现有 `共鸣矩阵` 显示 `星尘归航 +1 共鸣`。
- 已完成：归航重启本轮工坊，奖励固定 `1 共鸣`，保留共鸣、已启动永久节点、已领取共鸣里程碑和归航次数。
- 已完成：归航后有可用共鸣时，第 2 轮火花工作台会继续显示现有 `共鸣矩阵`，并优先提示“用共鸣启动永久节点，再推进下一轮工坊”。
- 已完成：两个永久节点已满时，阶段目标提示继续归航积累共鸣。
- 已完成：两个永久节点已满且仍有额外共鸣时，新一轮开局会说明“额外共鸣会保留到后续版本”，阶段目标也提示共鸣已暂存。
- 已完成：两个永久节点已满后继续归航，额外共鸣会点亮 `归航余辉`，让新一轮以少量星尘起步；余辉最高给 50 星尘，不引入新面板或存档字段。
- 已完成：`归航余辉` 现在会把起步星尘读回为可立即重建的自动采集器数量；50 星尘可立即重建前 3 台自动采集器。
- 已完成：已有归航历史且 2 个永久节点已启动后，现有 `共鸣矩阵` 会显示 `归航航线` 当前段、说明和下一段门槛；阶段目标优先读回当前航线。
- 已完成：未贯通 `归航航线` 时，现有 `共鸣矩阵` 和阶段目标会显示距 `稳航校准` / `深空归航` 还差几次归航、几点额外共鸣；贯通后转为长期储备读回。
- 已完成：`归航航线` 会把差距转成行动提示，区分“继续执行几次星尘归航”“额外共鸣已够，只补归航次数”“归航次数已够，只补额外共鸣”和“贯通后继续储备”。
- 已完成：`归航航线` 会预览下一段方向，说明达成当前门槛后进入 `稳航校准`、`深空归航`，或在三段贯通后继续储备后续版本。
- 已完成：`归航航线` 会读回当前段收益，说明额外共鸣已转成起步星尘、余辉开局已稳定并压缩重建时间，或三段贯通后进入后续版本储备。
- 已完成：即使余辉起步星尘已经被花掉，现有 `共鸣矩阵` 仍显示 `归航余辉` 已投入本轮重建节奏，不再退回“等待后续版本扩展用途”。
- 已完成：`归航航线` 会显示具名节奏预判，说明再归航几次是否已经足以自然进入 `稳航校准` / `深空归航`，避免把门槛读成两条互不相关的任务。
- 已完成：`归航航线` 会显示航线图，读回 `余辉起航 -> 稳航校准 -> 深空归航` 的完整三段路径和当前所在段。
- 已完成：`归航航线` 会在完整细节前显示航线摘要，把当前段、当前收益、距 `稳航校准` / `深空归航` 的差距和节奏预判压缩为首行扫读；当前收益会量化为“下轮起步可重建 N 台自动采集器”。
- 已完成：`归航航线` 会显示本段进度，把归航次数和额外共鸣读成当前值 / 目标值，例如 `归航 3/6 · 额外共鸣 2/4`；三段贯通后显示 `航线 3/3 已贯通`。
- 已完成：三段航线贯通后，`归航航线` 会显示当前长期储备数量，例如 `长期储备：6 次归航 / 5 点额外共鸣`，避免完成态只剩抽象“后续版本储备”。
- 已完成：阶段目标会把 `归航航线` 压成短航标读回，例如 `归航目标：稳航校准 2/3，补 3 次归航 / 2 点额外共鸣进入深空归航`；贯通后读回长期储备数量，完整行动说明继续留在矩阵详情中。
- 当前约束：不新增第三普通资源、新节点、节点等级树、任务系统、多生产线、额外面板或 telemetry。

## 当前 Gap

当前主要 gap 转向 `归航航线` 可读性复核：

- v0.6 已证明 50 星尘上限有明确节奏价值：可立刻重建前 3 台自动采集器。
- v0.7 已实现 `归航航线` 第一段、本段进度、航标化阶段目标、贯通态储备数量、具名动态差距读回、行动提示、下一段预览、当前收益读回、余辉已投入读回、具名节奏预判、航线图和带量化收益的航线摘要：用最多 3 个航线里程碑解释 20 小时后继续归航的方向，并告诉玩家当前段、归航次数 / 目标、额外共鸣 / 目标、下轮起步可重建几台自动采集器、距 `稳航校准` / `深空归航` 还差多少、下一步该做什么、继续归航是否足以补齐共鸣、达成后进入哪一段、完整三段路径在哪里，以及贯通后已经累计多少长期储备；余辉起步星尘被花掉后仍能读懂它已经生效。
- 下一次若继续扩张长期目标，应先复核带本段进度、航标化阶段目标、量化收益、具名差距和贯通态储备数量的航线是否真正减少迷失感，不能直接新增第三节点或节点等级树。
