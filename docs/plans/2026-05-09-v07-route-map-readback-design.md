# v0.7 归航航线图读回设计

## 背景

`归航航线` 已经显示当前段、下一段门槛、动态差距、行动提示、下一段预览、当前收益、余辉已投入读回和节奏预判。当前 gap 是：下一段预览只说明“达成后进入哪一段”，玩家仍要自己拼出完整三段长期路径。

## 设计

在 `ReturnRouteReadback` 中增加纯派生字段 `routeMap`：

- 第一段显示：`航线图：余辉起航 -> 稳航校准 -> 深空归航；当前位于余辉起航`
- 第二段显示当前位于 `稳航校准`
- 三段贯通后显示完整路径已贯通，后续归航进入长期储备

UI 继续复用现有 `共鸣矩阵` 的 `归航航线` 块，不新增面板、存档字段、资源、节点或 telemetry。阶段目标继续使用短行动提示，避免主目标行变长。

## 验证

- `src/return-route.test.ts` 覆盖第一段、第二段和三段贯通后的 `routeMap`。
- `src/App.test.tsx` 覆盖 SSR 中的航线图读回。
- 标准链路：`bun test`、`bun run test`、`bun run build`、`./ops/governor-check.sh`、`git diff --check`。
