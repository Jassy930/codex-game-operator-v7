# v0.7 归航航线价值摘要设计

## 背景

`归航航线` 已经在现有 `共鸣矩阵` 内显示当前段、差距、行动提示、节奏预判、航线图、当前收益和首行摘要。上一轮摘要解决了“在哪、差什么、按当前节奏能不能推进”，但第一行还没有说明当前段已经带来的价值。

这个缺口会让 20 小时后的重复归航仍像两项门槛清单，而不是一个正在生效的长期路线。

## 本轮选择

继续保持纯派生读回，只调整 `routeSummary`：

- 未贯通时，摘要包含当前段、当前价值短语、动态差距和节奏预判。
- 已贯通时，摘要继续读回三段贯通和长期储备。
- UI 仍复用现有 `共鸣矩阵` 的 `归航航线` 块，不新增行、不新增面板。

## 明确不做

- 不新增资源、节点、节点等级、任务系统、多生产线、额外面板、存档字段、视觉资产或 telemetry。
- 不修改 `星尘归航` 奖励。
- 不修改 `归航余辉` 公式或 50 星尘上限。
- 不改变 v0.7 最多 3 段航线的复杂度边界。
