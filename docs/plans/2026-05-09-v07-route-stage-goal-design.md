# v0.7 归航航线阶段目标航标设计

`归航航线` 已经能在现有 `共鸣矩阵` 里显示当前段、本段进度、量化收益、具名差距、行动提示、节奏预判、航线图、下一段预览和贯通态储备数量。复核发现：主阶段目标直接拼接完整 `actionHint`，对玩家扫读来说仍像长行动句。

## 设计

继续保持纯派生读回，只新增面向阶段目标的短文案：

- 未贯通时显示当前航线段、完成段数、进入下一段所需缺口，例如：`归航目标：稳航校准 2/3，补 3 次归航 / 2 点额外共鸣进入深空归航`。
- 如果只缺单一条件，目标行只显示该条件。
- 三段贯通后显示长期储备数量，例如：`归航目标：深空归航 3/3，长期储备 6 次归航 / 5 点额外共鸣`。
- `共鸣矩阵` 详情继续保留完整行动、收益、节奏预判、航线图和下一段预览，不新增面板、存档字段、资源、节点或 telemetry。
