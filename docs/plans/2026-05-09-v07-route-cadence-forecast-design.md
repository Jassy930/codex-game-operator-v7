# v0.7 归航航线节奏预判设计

## 背景

`归航航线` 已经显示当前段、下一段门槛、动态差距、行动提示、下一段预览和当前收益。当前 gap 是：归航会同时增加归航次数和额外共鸣，但现有行动提示把两者并列展示，玩家可能误读为两个互不相关的任务。

## 方案

在 `ReturnRouteReadback` 中增加 `cadenceForecast`：

- 未贯通且剩余归航次数足以自然补足目标共鸣时，显示“按当前路线再归航 N 次即可进入下一段”。
- 未贯通但归航次数补完后仍不够共鸣时，显示“再归航 N 次后，还需保留 M 点额外共鸣”。
- 归航次数已达标但额外共鸣不足时，显示“归航次数已达标，只差 M 点额外共鸣”。
- 三段贯通后，显示后续归航进入长期储备。

UI 复用现有 `共鸣矩阵` 的 `归航航线` 块，不新增面板、存档字段、资源、节点或 telemetry。阶段目标继续使用现有短行动提示，避免目标行过长。

## 验证
