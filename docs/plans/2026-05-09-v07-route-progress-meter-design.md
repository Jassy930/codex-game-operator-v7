# v0.7 归航航线本段进度设计

`归航航线` 已经能在现有 `共鸣矩阵` 中显示当前段、量化收益、具名差距、行动提示、节奏预判、航线图和下一段预览。复核发现：玩家能读懂“还差什么”，但仍要用下一段门槛反推当前归航次数和额外共鸣分别推进到了目标的几分之几。

## 设计

继续保持纯派生读回，只新增 `routeProgress`：

- 第一段显示：`本段进度：归航 2/3 · 额外共鸣 1/2`
- 第二段显示：`本段进度：归航 3/6 · 额外共鸣 2/4`
- 三段贯通后显示：`本段进度：航线 3/3 已贯通`

UI 仍复用现有 `共鸣矩阵` 的 `归航航线` 块，不新增面板、不新增存档字段、不新增资源或 telemetry。阶段目标继续保留短行动提示，避免目标行变长。

## 验证

- `src/return-route.test.ts` 覆盖第一段、第二段和贯通后的本段进度。
- `src/App.test.tsx` 覆盖 SSR 中的本段进度读回。
- 标准验证继续运行 `bun test`、`bun run test`、`bun run build`、`./ops/governor-check.sh` 和 `git diff --check`。
