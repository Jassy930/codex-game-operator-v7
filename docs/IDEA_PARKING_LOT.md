# Idea Parking Lot

Ideas are not backlog items.

An idea may only become implementation work after passing:

1. North Star check
2. Current stage check
3. Complexity budget
4. Research or feedback support, if needed
5. `DECISION.md` authorization

## Candidate Ideas

### Fast First Upgrade

Goal: first upgrade in 15-30 seconds.

### Next Goal Indicator

Goal: show what the player is saving for.

### Offline Progress

Goal: reward return visits.

### Soft Automation

Goal: let the player gradually reduce clicking.

### Milestone Unlock Preview

Goal: if future post-300s evidence shows stalled motivation, show a lightweight preview of the next meaningful milestone, such as 10 auto collectors or a later tuning threshold.

Constraints:
- not a current task
- no new resource
- no new panel
- no prestige
- no task system
- requires fresh feedback, local metrics, or self-playtest evidence before implementation
