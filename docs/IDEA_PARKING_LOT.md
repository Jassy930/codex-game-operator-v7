# Idea Parking Lot

Ideas are not backlog items.

An idea may only become implementation work after passing:

1. North Star check
2. Current stage check
3. Complexity budget
4. Research or feedback support, if needed
5. `DECISION.md` authorization

## Candidate Ideas

### Fast First Upgrade

Goal: first upgrade in 15-30 seconds.

### Next Goal Indicator

Goal: show what the player is saving for.

### Offline Progress

Goal: reward return visits.

### Soft Automation

Goal: let the player gradually reduce clicking.
